# Data Report

Use for weekly reports, business reviews, metric updates, experiments, and
operational summaries.

## Slide Shapes

- Executive summary: three to five findings, each tied to a number.
- KPI overview: stat callouts plus small trend chart.
- Detail chart: one chart with visible title, legend, and takeaway.
- Segment comparison: table or grouped bars.
- Funnel/process: stages with counts and conversion rates.
- Experiment/result: hypothesis, metric movement, decision.
- Risks/actions: concise table with owner/status if relevant.

## pptz Patterns

- Use chart `data` shorthand for category charts.
- Put exact values in table cells when the audience needs comparison; use charts
  when the audience needs trend or shape.
- Keep table styles compact and consistent across report slides.
- Use captions for source/date/measurement notes when needed.

## Avoid

- Tables that require reading every cell to understand the slide.
- Multiple unrelated chart types on one slide.
- Decorative charts without a stated takeaway.
